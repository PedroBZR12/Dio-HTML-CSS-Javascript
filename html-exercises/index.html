Explicações das tags!

Esse é um site que explica as tags mais comuns do HTML!

Aqui é o header! Onde colocamos as informações do cabeçalho, geralmente inclui tags de títulos como o h1, h2, h3, h4, h5 e/ou h6. O que diferencia essas tags é o tamanho do texto, o h1 é o maior e o h6 é o menor.

Também podemos colocar parágrafos, que são feitos com a tag p, que é a tag que você está lendo agora!

Explicação das tags dentro do main

Aqui é o main! Onde colocamos as informações principais do site, geralmente inclui tags de títulos como o h2, h3, h4, h5 e/ou h6.
